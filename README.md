## Тема 6 (Задание 6.1)

### Разработка комплекса коммуникационных средств для решения образовательных задач в рамках электронного учебного курса.

**1. Для реализации оперативной связи с обучающимися. Доведение информации о переносе занятий, новых материалах, опубликованных в курсе. Дистанционная консультации всей группы обучающихся.** Общая беседа в VK.

**2. Ответы на вопросы отдельным обучающимся.** Личные сообщения в VK/Почта. 

**3. Подход, стиль общения.** Стиль делового общения: демократический.

**4. Средства и виды взаимодействия.** Формальное взаимодействие. 

**5. Возможные трудности при взаимодействии:**

1. Игнорирование сообщений и информации.

2. Не выполнение заданий.

3. Сдача заданий в после дедлайна.

4. Не соблюдение делового этикета.

5. Конфликты. 

**6. Необходимые действия для разрешения трудностей:** 

1. Замечание и предупреждение (лично/через личные сообщения).

2. Снижение баллов за невыполненное задание в срок.
